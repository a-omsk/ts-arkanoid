# TypeScript Arkanoid

Hi! This is the simple arkanoid game built with TypeScript and HTML5 canvas. Use mouse or touchpad to control your "Vaus" paddle. Good luck!

# Requirements
Node >= 9.x

NPM > =5.x

# Install, build and run
```npm install```

```npm start``` 
    
Then open 127.0.0.1:3000 in your browser
